# 📈 Sales Forecasting using Machine Learning

> **Future Interns - Machine Learning Internship | Task 1**

## 📌 Project Overview

This project focuses on forecasting store sales using Machine Learning. Historical sales data was analyzed, cleaned, and transformed into meaningful features. A **Random Forest Regressor** was trained to predict future sales, helping businesses make data-driven inventory and demand planning decisions.

---

## 🎯 Objectives

- Analyze historical sales data
- Perform data preprocessing
- Engineer time-based features
- Train a Machine Learning model
- Evaluate prediction performance
- Generate future sales predictions
