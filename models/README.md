# Models

This folder is intended to store the trained machine learning model.

The model (`sales_forecasting_model.pkl`) was successfully generated during project execution.

Due to the GitHub web upload file size limitation (25 MB), the trained model file is not included in this repository.

To generate the model again:

1. Open `notebooks/Sales_Forecasting.ipynb`
2. Run all notebook cells.
3. The trained model will be saved automatically in the `models` folder.

Model Type: Random Forest Regressor

Frameworks Used:
- Python
- Pandas
- Scikit-learn
- Joblib
